# Prompt Engineering Kurs

Ovaj folder sadrži moje radne .ipynb fajlove i beleške iz Anthropic interaktivnog kursa za prompt inženjering.

## Sadržaj
- 01_Basic_Prompt_Structure (moj_rad.ipynb)
- 02_Being_Clear_and_Direct (zadaci.ipynb)
- 03_Assigning_Roles (eksperiment.ipynb)
- Beleške (primeri.md, prompt-sazetak.txt)

## Pokretanje
1. Instaliraj Python 3.11+
2. `pip install notebook`
3. Pokreni: `jupyter notebook`
4. Otvori željeni fajl iz foldera
